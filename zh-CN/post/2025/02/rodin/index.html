发表于 2025-02-01 Disqus：

在 macOS 上安装 Rodin 的步骤（M1 ARM 架构）：

  1. 安装 x86 JDK

    • 从 Oracle 下载 Intel x86 JDK 18（macOS .dmg）：

      jdk-18.0.2.1_macos-x64_bin.dmg

    • 运行安装程序，它会自动配置 Java 环境。

  2. 下载并安装 Rodin

    • 从 SourceForge 获取 Rodin 3.8/3.9（x86_64 版本）：

      rodin-3.8.0.202304051545-macosx.cocoa.x86_64.tar.gz

    • 解压 .tar.gz 文件，并将 Rodin.app 移动到 /Applications 文件夹。

  3. 修复 macOS 安全权限
