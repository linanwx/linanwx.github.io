Introduction

The latest version of Windows 10 system’s built-in Pinyin input method already supports Xiaohe Shuangpin. Mom no longer has to worry about me installing bloatware!

After 4 years, here’s an updated advanced setup method. Mom no longer has to worry about me configuring keys one by one:

  1. Press Win + R, type regedit, and open Registry Editor
  2. Find the Computer\HKEY_CURRENT_USER\Software\Microsoft\InputMethod\Settings\CHS entry

  4. Open Microsoft Pinyin input method settings and set Xiaohe Shuangpin as the default scheme for Shuangpin.

If you’re concerned about registry modifications causing system issues, here’s a manual configuration method:

Step 1: Open Settings

Although the system’s Pinyin input method doesn’t have Xiaohe Shuangpin by default, we can customize it.

First, find the Shuangpin settings location. If you can’t find it, you can refer to the animated GIF:

Step 2: Edit Custom Shuangpin

The Shuangpin settings are shown in the figure below

Except for the template section which doesn’t need to be set, the rest should be configured according to the image. One is the zero initial mode, and the other is custom key mapping. Custom key mapping should be set according to the preview in the image. Note that there is no “er” syllable in Xiaohe Shuangpin, so you can set it to either the “e” or “r” key. Also, both the “ue” and “ve” syllables are set to the “T” key. Here’s a tip: when setting keys, you can directly press the corresponding key instead of finding the key to set.

Step 3: Save the Scheme

If the add button becomes available, it means the scheme is set correctly. After adding the scheme, set it as the default scheme, and you can now use Xiaohe Shuangpin!
